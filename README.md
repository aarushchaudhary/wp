## ⚠️ Note for AngularJS Examples

In AngularJS files, replace the local script reference:

```html
<script src="./angular-1.8.2/angular.min.js"></script>
```

with the Google CDN version:

```html
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.2/angular.min.js"></script>
```

Using the CDN ensures the correct AngularJS version loads without needing the local library.
